# Flutter Quiz App

A fun and engaging quiz application built with Flutter. Test your knowledge across various categories and difficulty levels, track your high scores, and learn from your mistakes with an answer review system.

## üöÄ Features

*   **Dynamic Theming:** Seamlessly switch between light and dark modes.
*   **Multiple Categories:** Choose from a wide range of topics like History, Science, Movies, and more.
*   **Difficulty Levels:** Select from Easy, Medium, or Hard to match your skill level.
*   **Customizable Timers:** Challenge yourself with selectable time limits (2, 5, 10 minutes) or play at your own pace with an unlimited option.
*   **Lives System:** You get 3 lives per quiz. An incorrect answer costs a life. Lose all three, and the quiz ends!
*   **Streak Counter:** Build up a fire streak by answering questions correctly in a row.
*   **High Score Tracking:** The app saves your best score for each category and difficulty combination, displayed right on the start screen.
*   **Answer Review:** After finishing a quiz, you can go through a detailed review of each question, showing your answer, the correct answer, and a full explanation.
*   **Polished UI:** Features a beautiful splash screen and smooth animations for question transitions.
*   **Settings Screen:** A dedicated screen to manage app settings, including:
    *   Theme selection (Light, Dark, System).
    *   An option to reset all high scores.

## üèÅ Getting Started

To get a local copy up and running, follow these simple steps.

1.  **Clone the repository**
    ```sh
    git clone <your-repository-url>
    ```
2.  **Navigate to the project directory**
    ```sh
    cd quiz_app
    ```
3.  **Install dependencies**
    ```sh
    flutter pub get
    ```
4.  **Run the app**
    ```sh
    flutter run
    ```
